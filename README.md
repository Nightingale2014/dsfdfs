---

## ✨ 주요 기능

- **검색창 구현**: 사용자 입력을 통한 태그 추가 가능
- **태그 리스트 관리**: 입력한 태그 삭제 가능
- **기본 태그 제공**: 수능공부, 자격증준비, 공무원시험 등
- **중앙 정렬**: 전체 화면 중앙에 정렬하여 보기 편한 UI 구성
- **돋보기 버튼**: 하얀색 라인형 돋보기 아이콘 적용
- **반응형**: 검색창과 태그 영역 모두 깔끔한 정렬 유지
